## ADDED Requirements

### Requirement: マップ操作モードの切り替え
マップには `edit`（エリア選択）と `pan`（地図移動）の2つの操作モードがあり、ユーザーはいつでも切り替えられなければならない（SHALL）。デフォルトは `edit` モードとする。

#### Scenario: 初期状態は edit モード
- **WHEN** コレクションページを開く
- **THEN** マップは edit モードで表示され、エリアをタップ/クリックするとモーダルが開く

#### Scenario: pan モードへの切り替え
- **WHEN** ユーザーがモード切り替えボタン（✋アイコン）を押す
- **THEN** マップが pan モードに切り替わり、ボタンアイコンが ↖ に変わる

#### Scenario: edit モードへの切り替え
- **WHEN** ユーザーが pan モード中にモード切り替えボタン（↖アイコン）を押す
- **THEN** マップが edit モードに切り替わり、ボタンアイコンが ✋ に変わる

#### Scenario: モード切り替えボタンの位置
- **WHEN** コレクションページが表示される
- **THEN** モード切り替えボタンは地図の左下に常時表示される
